## Общее описание
Инструмент для визуализации графа зависимостей пакетов Rust (Cargo). Реализован как CLI-приложение с поддержкой тестового режима и фильтрации пакетов.

## Функции и настройки
- **Источник параметров**: опции командной строки
- **Настраиваемые параметры**:
  - Имя анализируемого пакета
  - URL репозитория или путь к тестовому файлу
  - Режим работы с тестовым репозиторием
  - Версия пакета
  - Имя файла для изображения графа
  - Подстрока для фильтрации пакетов
